B.     \(\frac{{v\cos \theta }}{g}\)

C.     \(\frac{{2v\sin \theta }}{g}\)

Markscheme

D

Examiners report

Clearly when \(\theta  = 90^\circ ,{\text{ }}t = 0\), so A and C must be incorrect. The relevant equation of motion relating s, v, a, t includes \(\frac{1}{2}\), so D must be the correct response.
